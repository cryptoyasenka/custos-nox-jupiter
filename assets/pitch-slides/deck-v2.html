Zero Friction Adoption
06 / 09
From vulnerable to protected
in 5 minutes
No Rust · No credit card · No configuration hell
1
Copy your Squads PDA
It's visible in app.squads.so. Just an address — same as a wallet address.
2
Set one env variable
CUSTOS_WATCH=<PROGRAM>:<YOUR_PDA>
Pick any sink: Discord, Slack, Telegram, or stdout JSON — that's the full config.
3
Run npm run dev
Or docker run. From that moment — any config change on your multisig fires an immediate alert.
No paid tiers ever. Public Goods Award sustainability — covered by Solana Foundation grant + Superteam Ukraine. Free Helius RPC, MIT licensed, public good permanently.
